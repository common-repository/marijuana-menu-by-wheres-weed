Add your Where's Weed business menu to your wordpress site.

== Description ==

WheresWeed.com is a marijuana dispensary locator service. This plugin allows our businesses to embed their Where's Weed menu directly on their own Wordpress site. Updates are pulled in real-time from your Where's Weed profile.

Learn more and read a detailed tutorial at: https://biz.wheresweed.com/2015/nov/embedding-your-wheres-weed-marijuana-menu-on-your-wordpress-site

== Installation ==

1. Upload the uncompressed zip file into the `/wp-content/plugins/` directory, or, upload the zip file through the plugins.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Place your business url (e.g. https://wheresweed.com/state/city/business-name) in the plugin settings
4. Place the shortcode "[ww_menu]" (without quotation marks) in a wordpress page wherever you would like the menu to be displayed.

== Frequently Asked Questions ==

= Does this plugin update in real-time? =

Yes, updates are pulled in real-time directly form your Where's Weed profile.

= My menu isn't displaying, what's wrong? = 

Be sure you entered in a valid business profile URL in the plugin settings. If you are still experiencing problems, please contact us at https://wheresweed.com/contact.php

= Can I adjust the styling of the menu? =

Yes you can! Please contact us for menu customization!
